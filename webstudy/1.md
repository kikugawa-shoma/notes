# ローカルでのwebサーバの起動

## ディレクトリ構成
```
C:.
└─test
    index.html
    test.py
```

## コマンド
```
C:\Users\UserName>cd programming\web\test

C:\Users\UserName\programming\web\test>python -m http.server --cgi
Serving HTTP on 0.0.0.0 port 8000 (http://0.0.0.0:8000/) ...
```
上のコマンドを実行したうえでブラウザでhttp://localhost:8000/にアクセス
